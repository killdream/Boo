Boo

Prototypical utilities for inheritance in JavaScript

Overview

Boo —at a glance—

Boo is a minimal library to provide utilities for handling prototypical inheritance in JavaScript in a sane way.

You can grab the lastest release, or clone the entire Github repository:

Why Use It

JavaScript has a nice prototypical inheritance system, but its implementation is just broken. Boo abstract the bad bits of this implementation in a neat package, while maintaining all the sweetness of prototypes.

» Other reasons to use Boo

Supported platforms

Boo should work in any ECMAScript 5-compliant environment — like Node.js, Chrome and Firefox. Just load it and hack away.
