Boo

Supported Platforms

What do you need to run Boo

Boo should work in any ECMAScript 5-compliant environment — like Node.js, Chrome and Firefox. Just load it and hack away.

Boo depends on the functional Array additions from JavaScript 1.6, plus Object.getPrototypeOf, Object.keys and Object.create.
