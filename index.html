Boo

Boo takes the “scary” away from JavaScript's OO by providing you with the right prototypical primitives for composing and extending objects.
  • Node.js
  • Browser

Installing

$ npm install boo

Usage

var boo = require('boo')

var Shape = boo.Base.derive({
  init: function(colour) {
    this.colour = colour }})

var Rect = Shape.derive({
  init: function(x, y, width, height, colour) {
    Shape.init.call(this, colour)
    this.x = x;         this.y = y
    this.width = width; this.height = height }})

var r1 = Rect.make(0, 0, 50, 50, 'crimson')
// => { x: 0, y: 0, width: 50, height: 50, colour: 'crimson' }

Installing

Add the script tag just before </body> and the script that depends on Boo.

<script src="/path/to/boo.js"></script>
